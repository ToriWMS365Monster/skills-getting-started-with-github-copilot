## Running the test suite 🧪

Backend tests live in a `tests/` directory and use `pytest` with FastAPI's
`TestClient`. After installing dependencies (including `pytest`), simply run:

```bash
pip install -r requirements.txt
pytest
```

All tests should pass and verify core API behavior including successful
signups/removals and the associated error cases.
